Admin

Staff sign-in. Configure AUTH_SECRET, ADMIN_EMAIL, and ADMIN_PASSWORD_HASH (bcrypt) in Vercel project settings.

Generate a hash locally: node -e "require('bcryptjs').hash('your-password',10).then(console.log)" and paste the result into ADMIN_PASSWORD_HASH.
